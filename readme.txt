Enhance Webdzier Themes WordPress Themes additional functionality.

== Description ==

This plugin is developed for webdzier themes. This plugin creates the additional sections on the front page in webdzier themes.
Webdzier Companion is a plugin creation to enhance the functionality of WordPress theme created by webdzier.

== Features ==

Slider Section
About Section
Service Section
Info Section
Call To Action Section 
And many more.
